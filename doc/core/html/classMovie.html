Detailed Description

Represents a concrete Item with movies informations.

The Movie class is a part of the composite pattern used to store the Item tree. It is derivated from the Item class and represents a movie with specific movie informations : summary and notation.

Note:
Movie class doesn't override children management method because a Movie is a basic Item and can not have children.
