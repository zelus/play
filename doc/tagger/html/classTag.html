Detailed Description

template<typename T>
class Tag< T >

Represents a simple Tag (key-word)

This class is managed by the TagManager class : it shouldn't be manipulated directly (it may break the Tag based search engine).

Note:
Tag class is a template class and need a special item interface to work properly : the tagged item must provide a public getId() method returning a const string&. The return of this method should be an unique identifier of the tagged object.
Warning:
Tag class only works with pointers (Item* instead of Item).
